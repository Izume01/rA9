# RA9

Autonomous coding agent for the terminal.

RA9 is a stateful AI runtime designed for real development workflows — not just chat-based code generation.

It combines:

- Persistent PTY sessions
- Context-aware memory
- Goal planning
- Tool orchestration
- Safe command execution
- Multi-agent workflows

into a single terminal-native experience.

---

## Features

- Stateful shell sessions
- File editing + patching
- Persistent memory
- Task planning + decomposition
- Git integration
- Multi-agent execution
- Command safety layer
- Tool/plugin system
- Streaming terminal UI

---

## Vision

RA9 is designed to feel less like a chatbot and more like an operating layer for autonomous software development.
